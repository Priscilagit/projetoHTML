HTML

(Voltar)

O HTML é uma Linguagem de Marcação de HiperTexto. Ele possuem uma estrutura como um bloco de construção mais básico da web. Que define o significado e a estrutura do conteúdo da web. Sua estrutura é composto de uma série de elelmentos, que você usa para delimitar ou agrupar diferentes partes do conteúdo oara que ele pareça ou atue de determinada maneira.

Sua estrutura

(Voltar)

O HTML tem uma estrutura composta pelas tags HTML, HEAD e BODY, além da intrução !DOCTYPE. A tag HTML é onde começa a estrututra. O cabeçalho é composta pela tags HEAD, onde contém o título da página. Já o corpo é composto pela tags BODY, onde contém o conteúdo da página a parte visivel do usuário.

Tags, como usa-las?

(Voltar)

As tags HTML deve conter 3 partes: Uma tag de abertura que tem como símbolo < >, o conteúdo que são introduções curtas para exibir o elemento da página e uma tag de fechamento que termina com o símbolo < / >. Mas algumas tags podem ser abertas, isso isgnifica que não precisa de seu fechamento. Como citado acima, as tags que começam a estruturar o HTML são :

  • HTML, que vai ser onde ficará o começa da estrutura e também onde podemos colocar as âncoragens de imagens, estilos e referências do conteúdo.
  • HEAD, onde podemos colocar o título da página.
  • BODY, onde é o corpo da estrutura. Dentro do BODY é declarada outras tags como:
    • p: tag de um parágrafo, onde formata qualquer texto entre a tag de abertura < > e a de fechamento < / > .
    • h1: tag de título, onde é composta por h1(que é um título), entre o h2 até o h6 são considerados subtítulos. Onde o tamanho de cada um é diminuido conforme é usado.
    • div: tag de divisão, geralmanto ela divide a estrutura dentro do BODY como blocos.
    • a: tag de âncora, ele é usada para âncorar um link.
    • ul: tag de lista não ordenada, onde a lista é exibida por pontos quando declarada.
    • ol: tag de lista ordenada, onde a lista será exibida por ordem numérica.
    • li: tag de lista, como o nome já diz será exibido uma lista que pode ser ordenada ou não ordenada.
    • img: tag de imagem, onde você pode colocar uma imagem com um link par ser exibida na página
Além dessas tags que são as mais básica de uma estrutura HTML, temos também as tags de estilização, onde podemos mudar o estilos do texto, palavra ou até mesmo da página.
  • style: tag de estilos, onde você pode mudar o tamanho, cor ou fonte do conteúdo.
  • b: tag de negrito, irá formatar qualquer texto ou palavra em negrito.
  • i: tag de itálico, formatará o texto em itálico.
  • u: tag de sublinhado, sublinha o texto ou palavra.
Entre outros.
